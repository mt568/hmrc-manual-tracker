When a repayment is issued through the SA system a notice, detailing the make up of the repayment, is also issued. A copy notice is issued to the taxpayer / agent / nominee, as appropriate.

The notice will show the following

* Name of the Officer in Charge
* Office name / address / telephone number
* Date of issue of the repayment
* Taxpayer’s Unique Taxpayer Reference (UTR) and repayment serial number
* Payee’s name and address
* Capacity name (if applicable)
* Capacity role (if applicable)
* Taxpayer’s name (if capacity acting and taxpayer is not the payee)
* Client reference (if applicable)

The messages that will be issued in particular circumstances are detailed below

Note: From November 2010, the first 5 characters of the account number, or building society roll number (where applicable), were replaced by ‘XXXXX’.

#### Circumstance

Repayment to taxpayer by Payable Order

* No agent
* No repayment supplement

#### Repayment message

Thank you for your recent claim to repayment

A payable order for £…… is attached

If you have any queries please contact this office, quoting the reference shown above

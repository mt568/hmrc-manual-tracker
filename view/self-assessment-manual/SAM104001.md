title: 'Records: trace taxpayer record: introduction'
---

In day to day work, you may want to trace a taxpayer record when

* Engaged on routine post tracing
* Dealing with telephone enquiries
* You have responsibility for either

  + Setting up new SA records

  Or

  + Transfer of records

To do this you can use either the TBS function FIND TAXPAYER or the NPS function TRACE AND MATCH.

The NPS function can be used to search for individual and employer records and the TBS function can be used to trace only for individual records.

Guidance on using the TBS function can be found in the Taxpayer Business Service Guide (TBSG) at TBSG032 and guidance on using the NPS function can be found in the PAYE Manual at PAYE120001 onwards.

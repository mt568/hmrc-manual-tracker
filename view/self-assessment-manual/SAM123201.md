If you have to unlog an unsatisfactory return follow steps 1 - 5 below. For details of how to access the SA functions, select ‘Index Of Functions’ on the left of the screen.

|  |  |
| --- | --- |
| 1. | Use function MAINTAIN RETURN SUMMARY to check the return has been logged |
|  |  |
| 2. | If the return has been logged use function UNLOG RETURN to unlog the return and select a reason for the unlogging from the drop down menu |
| 3. | If the whole return is to be sent back to the trustee and some of the details have been entered in TSS, use the TSS to delete them |
| 4. | Cross through any date stamp on the front of the return before returning the form |
| 5. | Keep a copy of the letter you send to the trustee/agent concerning the unsatisfactory return in the trust’s file (This content has been withheld because of exemptions in the Freedom of Information Act 2000) return received on (date) unlogged on (date)) from the SEES Notes Paster |

title: 'Records: set up taxpayer record: trust record and partner source (Action Guide)'

When a trust becomes a partner in a partnership, follow steps 1 - 6 below.

For details of how to access any of the SA functions, select ‘Index Of Functions’ on the left of the screen.

|  |  |
| --- | --- |
| 1. | Decide which trustee or executor name you are going to use on the SA record. Normally this will be the main or first named |
|  |  |
| 2. | Ensure that you have a note of the partnership Unique Taxpayer Reference (UTR) and the date that the trust commenced as a partner |
| 3. | Set up an individual SA record in accordance with Action Guide ‘Set Up Individual Record’ ([SAM100221](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam100221)), subject to the following advice |
|  | \* Enter the trustee’s or executor’s name in the Name field |
|  | \* In the Address field, enter the trustee’s or executor’s address |
|  | \* Do not enter the trustee’s or executor’s NINO |
| 4. | Use function AMEND TAXPAYER SIGNALS to |
|  | \* Set the Last SA Return Required For Year Ending 5 April signal to the 5 April previous to the date the partner source commenced |
|  | \* Set the No Repayment signal |
|  | \* Enter the trust UTR and the trustee’s or executor’s NINO in the Free Format Notes field |
|  | \* Set the SA Entry / Exit Letter Inhibited signal to ‘Y’ |
| 5. | Use function SET UP PARTNER to create a link between the nominated trustee’s or executor’s individual record and the partnership record |
| 6. | Make an SA Note on the partnership record of the name, address and UTR of the record that you have linked to the partnership record and name, address and UTR of the trust |

Where you want to add a new Permanent Note follow steps 1 - 3 below.

1. Use function MAINTAIN PERMANENT NOTES to view the Permanent Notes already recorded
2. On the view screen

```
* Select the [Add Note] button
```

1. On the New Permanent Note screen

```
* After selecting the relevant Permanent Note from SEES Notes Paster
* Select the [EDIT] button
* Select the [PASTE] buttonYour Permanent Note will now be recorded.
```

|  |  |
| --- | --- |
| 1. | Check through the sets of SAI received and determine which is for the business with the largest turnover |
|  |  |
| 2. | Check to see whether, for this business, there is SAI for more than one period of account. Determine which set covers the latest period |
| 3. | Capture the SAI for the latest period of account only, using function CAPTURE RETURN |

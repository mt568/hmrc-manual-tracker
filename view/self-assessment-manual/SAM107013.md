When you are notified that Welsh language correspondence is no longer required, follow steps 1 - 8 below.

For details of how to access any of the SA functions, select ‘Index Of Functions’ on the left of the screen.

For ‘Individual’ type taxpayer records

1. Use TBS function AMEND TAXPAYER DESIGNATORY DETAILS to

```
* Set the Language signal to ‘English’
```

1. Use function AMEND TAXPAYER SIGNALS to

```
* Review the Free Format Notes field
* Make a note of the communication address or main capacity address shown, then delete that part of the note
```

1. If a communication address is shown, use TBS function AMEND TAXPAYER ADDRESS to amend the nominated address to the address which was within SA Notes. This will update the communication address in SA automatically
2. If a main capacity address is shown, and the capacity address was amended to the Welsh Language Unit, use TBS function AMEND CAPACITOR DETAILS to update the capacity address to the details held in SA Notes

For Non Resident Company Landlord and taxpayer types other than ‘Individual’

1. Use function AMEND TAXPAYER SIGNALS to

```
* Set the Language signal to ‘English’
* Review the Free Format Notes field
* Make a note of the communication address or main capacity address shown, then delete that part of the note
* If a main capacity address is shown, follow step 6
* If a communication address is shown, follow step 7
* If no address is shown, follow step 8
```

Main capacity address shown

1. Use function MAINTAIN CAPACITY

```
* Select the [New Address] button. You will be taken to function STORE ADDRESS
* In function STORE ADDRESS, amend the address of the main capacity, from the Welsh Language Unit to that noted from the Free Format Notes field
```

Communication address shown

1. Use function ADD/AMEND COMMUNICATION NAME AND ADDRESS

```
* Amend the communication address in the Communication Address field, from the Welsh Language Unit to that noted from the Free Format Notes field
```

No address shown

1. Use function ADD/AMEND COMMUNICATION NAME AND ADDRESS to

```
* Delete the entry for the Welsh Language Unit in the Communication Address field
```

The Visually Impaired Media Unit (VIMU) provides information in alternative formats for customers with impaired vision.

The alternative formats provided are

* Braille
* Audio
* Large print

Special output signals can be set in Function SET UP INDIVIDUAL TAXPAYER to indicate that output is required in these formats. For cases already set up, the signals can be set using TBS Function AMEND TAXPAYER DESIGNATORY DETAILS.

Setting the special output signal will

* Set the communication address to the VIMU

> And

* Ensure that any future update to the CID framework does not alter the communication address that is stored on the ITSA computer system

Where a customer requests an output in an alternative format, follow the Ocelot guidance: Sending a Visually Impaired request form - Step 1 to send a request form to the VIMU.

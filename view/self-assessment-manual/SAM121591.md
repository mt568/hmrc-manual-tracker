When you have entered as much information as possible from the return follow steps 1 - 8 below.

For details of how to access the SA function, select ‘Index Of Functions’ on the left of the screen.

## Where you are able to capture all the return information

1. View the calculation. Check that the end result is as expected

2. Save the return as fully captured in function CAPTURE RETURN by selecting the [Full] button on the SAVE RETURN screen when you close or exit the return

3. File the return away

## Where you are only able to capture part of the return information

4. Save the return as part captured in function CAPTURE RETURN by selecting the [Part] button on the SAVE RETURN screen when you close or exit the return

5. Ask for any outstanding information. For example from a colleague or the taxpayer

6. Make a note on the return if you decide that the return is not unsatisfactory but send part back to the taxpayer

7. Place the papers on B/F for the outstanding information

8. On receipt of the information

* Use function CAPTURE RETURN to capture the remaining details
* Save the return as fully captured by selecting the [Full] button on the SAVE RETURN screen when you close or exit the return

Note: When the reason for the entry on the ‘Manual Issue of Return’ work list is ‘Welsh Language Signal’, the necessary action to deal with the work item will be taken by the Welsh Contact Centre. Full details for that office are

HMRC Welsh Customer Service  
HM Revenue & Customs  
Thedford House  
High Street  
PORTHMADOG  
LL49 9BF

Telephone number 0300 200 1900

Where the reason for the entry on the ‘Manual Issue of Return’ work list is ‘WELSH LANGUAGE SIGNAL’, follow steps 1 - 7 below.

For details of how to access any of the SA functions, select ‘Index Of Functions’ on the left of the screen.

|  |  |
| --- | --- |
| 1. | Establish the address, steps 2 - 4, to which the return is to be sent (there should be either a capacity address or communication address present on the SA record) |
|  |  |
| 2. | Use function MAINTAIN CAPACITY (available direct from the function menu) |
|  | \* If a capacity address is present, and it is the Welsh Unit |
|  | \* Use function AMEND TAXPAYER SIGNALS to view the Free Format Notes field |
|  | \* Use the address shown in the Free Format Notes field |
|  | \* If a capacity address is present, and it is not the Welsh Unit |
|  | \* Use that address |
|  | \* Follow step 4. The SA record has to be updated |
|  | \* If no capacity address is present, follow step 3 |
| 3. | Use function VIEW COMMUNICATION NAME AND ADDRESS |
|  | \* If a communication address is present, and it is the Welsh Unit |
|  | \* Use function AMEND TAXPAYER SIGNALS to view the Free Format Notes field |
|  | \* Use the address shown in the Free Format Notes field |
|  | \* If a communication address is present, and it is not the Welsh Unit |
|  | \* Use the address shown |
|  | \* Follow step 4. The SA record has to be updated |
|  | \* If, exceptionally, neither capacity address nor communication address is present, the Welsh Contact Centre will follow step 4 |
| 4. | Review the papers, in particular the correspondence that included the request for Welsh language correspondence |
|  | \* When you have established the correct address, follow the Action Guide in section ‘Welsh Language’, in business area ‘Records’, to update the SA record |
| 5. | Prepare a Welsh language version of the return. Use function MAINTAIN RETURN PROFILE to establish whether a Main Return (SA100) or Short Tax Return (SA200) is required |
|  | \* Address it to the taxpayer, or (main) person or body appointed to act at the address that you have established |
|  | \* Use function MAINTAIN RETURN PROFILE to establish the supplementary pages that are required with the return |
| 6. | If you have prepared the return for annual bulk issue, hold the return with other returns to be issued manually. You may want to |
|  | \* Use function SET B/F DATE to b/f the work item and make a B/F Note of the action taken |
|  | \* Prior to issue, use function RECORD DATE OF CLERICAL ISSUE to enter the date of issue on the SA record (the function allows you to enter a date up to 5 days later than today’s date). The work item will be deleted automatically |
| 7. | If you have prepared the return after 5 April in the return year |
|  | \* Use function RECORD DATE OF CLERICAL ISSUE to enter the date of issue on the SA record. The work item will be deleted automatically |

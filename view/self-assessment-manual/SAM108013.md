title: 'Records: SA and permanent notes: amending a permanent  note (Action Guide)'
---

Where you want to amend a permanent note follow steps 1 - 3 below

1. Use function MAINTAIN PERMANENT NOTES to view the permanent notes already recorded
2. Select the line number of the permanent note you want to amend, then

```
* Select the [Amend Note] button
```

1. On the Amend Permanent Note screen

```
* Amend the text of the permanent note
* Select the [OK] button
```

title: 'Returns: filing returns online: filing returns online and  work management'

Following acceptance of returns information sent online the case may be entered on the ‘Daily Returns Review: Electronic Return Cases’ work list. The reason for the work item will be shown as INT / ADC / PAYE CIC.

In this instance a coding (integrity) check is required.

Detailed advice on the action to take to clear entries (work items) appearing on the ‘Daily Returns Review: Electronic Return Cases’ work list is available in the section ‘Returns Work Lists’ at [SAM125020](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam125020).

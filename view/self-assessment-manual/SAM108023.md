An SA Note generated automatically cannot be amended.

Where you want to amend an SA Note follow steps 1 - 3 below

|  |  |
| --- | --- |
| 1. | Use function MAINTAIN SA NOTES to view the SA Notes already recorded |
|  |  |
| 2. | Select the line number of the SA Note you want to amend, then |
|  | \* Select the [Amend Note] button |
| 3. | On the Amended SA Note screen |
|  | \* Amend the text of the SA Note |
|  | \* Select the [OK] button. The amended SA Note is recorded together with your PID and the date the amendment was made |

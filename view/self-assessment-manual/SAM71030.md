title: 'Manage work: work management: how work items are  created'
---

When certain events occur in the SA system, details are passed to Work Management to create a work item.

The trigger event(s) differ for each work item.

Some of the details apply only to specific work items. For example

* The amount of a debt
* The number of returns outstandingThe following details are common to all work items
* UTR
* Taxpayer name
* Date the work item was createdOther details, although potentially common to all work items, will not apply to every item. For example
* Employers reference (EMPREF)
* NINO
* Schedule D register numberThe Index of Work Lists in the left bar provides more information.

title: 'Records: SA and permanent notes: adding a new SA note (Action Guide)'
---

Where you want to add a new SA Note, follow steps 1 - 3 below.

For details of how to access the SA function, select ‘Index Of Functions’ on the left of the screen.

1.  Use function MAINTAIN SA NOTES to view the SA Notes already recorded

2.  On the view screen

* Select the [Add Note] button

3.  On the New SA Note screen

If you are adding a note obtained from the SEES Notes Paster

* After selecting the relevant SA Note from the SEES Notes Paster
* Select the [EDIT] button
* Select the [PASTE] button

If you are adding a free format SA Note

* Enter the new SA Note
* Select the [OK] button

If you wish to add a second SA note, as a continuation of the first (see [SAM108020](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam108020))

* Indicate at the start of the second note that it is a continuation of the previous SA note

Your new SA Note is now recorded

**NOTE**: there may be cases where the New SA Note
button is ‘greyed out’, and you cannot add a note for a case. This is usually
where a record is restricted, such as in some Wealthy Cases, where read-only
access is given, or some Overseas cases, where it may be necessary to be in a
different User Role in order to update the record. These cases should be
referred to a local SPOC or Tech for advice about how to proceed.

A work item may be created on the ‘Weekly Returns Review’ Work List for any one, or more, of three reasons. Where there is more than one reason, there will still only be one work item. You will be able to see the reasons by using function SELECTED CASE. For ease of reference, separate Information and Action Guides are available for each of those reasons.

See subjects and associated Action Guides

* Weekly Returns Review: Liability Unlikely ([SAM125180](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam125180))
* Weekly Returns Review: Source Exception ([SAM125190](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam125190))
* Weekly Returns Review: Transfer Of Allowances ([SAM125200](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam125200))

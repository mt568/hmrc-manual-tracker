When notification of death was received, the action taken should have included the following

* Manual Return signal set
* LAST SA RETURN REQUIRED FOR YEAR ENDING 5 APRIL signal set to the end of the year in which death occurred
* Deceased signal set
* Date of death entered on the SA record
* A capacity role of executor or administrator set up, if appropriate

You should now

* Refer to [SAM125081](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam125081) to decide if a return to the date of death is required
* If so, prepare a return manually and use function RECORD DATE OF CLERICAL ISSUE to enter the date of issue on the SA record (the function allows you to enter a date up to 5 days later than today’s date). Note: The return must be addressed to a named person to be considered correctly served. For example ‘Mr A N Other as Personal Representative of ……….. Deceased’

When certain events occur in the SA system, details are passed to Work Management to create a work item. Work Management then

* Allocates that work item to the office with responsibility for that type of HMRC activity

And

* Enters that work item on the appropriate work list, which is available for display on screen

Access to a work list is from the operator menu, either

* Direct

Or

* Through CASE SUMMARY

## Case Summary

In CASE SUMMARY the [Work] button will be present only if there is currently at least one open work item, and you have access to the appropriate Work Management function. The button will take you to function TAXPAYER WORK LIST. This function allows you to view a summary of work lists, for all types of Revenue activity, containing an entry for the selected taxpayer. It will not allow you to access any work list shown on screen that has not been allocated to your Organisation Unit, and is not appropriate to your user role.

For example, if your office has processing responsibility for the taxpayer, your access is restricted to the Processing work lists under your user role.

When accessing a particular work list, you will be presented with a list of all work items

* Of the same work type
* For your Organisation Unit
* Under your user role

## Menu

From the drop down menu at the top of the screen, you may access the following functions, which summarise work lists with current entries allocated to your Organisation Unit under your user role

* OPERATOR WORK LIST SUMMARY
* IMMEDIATE ACTION WORK LISTS

On selecting a work list

* Where you have update access to that work list, the full range of Work Management functions is available to you
* Where you do not have update access, the more limited functions available allow you to

  + View the entries on the selected work list
  + Filter and Sort that list to provide a personal view of it
  + Print the list
  + View all available details on a selected work item

If you have created buttons for frequently accessed functions and work lists, you will select the appropriate button to access the required function or work list.

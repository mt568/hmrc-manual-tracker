Following capture of the return where payments on account are not required under the 80 per cent test, follow steps 1 - 7 below.

For details of how to access the SA functions, select ‘Index Of Functions’ on the left of the screen.

This guide is presented as follows

[Payments on account to be set to zero for the year following the return year and a return for that year is not recorded as issued. (The annual selection process has not yet taken place)](#IDAVJ33H) - Steps 1 - 4  
[Payments on account to be set to zero for the year following the return year but a return for that year is recorded as issued](#IDALM33H) - Steps 5 - 7

## Payments on account to be set to zero for the year following the return year and a return for that year is not recorded as issued. (The annual selection process has not yet taken place)

1. Use function AMEND TAXPAYER SIGNALS to enter the year of the return in the Last SA Return Required for Year Ending 5th April field. For example if payments on account are not due for 2007/08 following capture of the 2006/07 return, enter ‘07’

2. Access function CREATE RETURN CHARGE for the return year and select the [OK] button. This will remove the payments on account

3. Use function AMEND TAXPAYER SIGNALS to delete the entry in the Last SA Return Required for Year Ending 5th April field

4. Make an SA Note to check the payments on account position if the return is subsequently amended

Note: If an amendment is made to the return and payments on account are created but are not required under the 80 per cent test, repeat the above action

## Payments on account to be set to zero for the year following the return year but a return for that year is recorded as issued

Using the SA return, for example where payments on account for 2008/09 to be set to zero you should be using the 2007/08 return

5. Add together the figures from boxes A185, A156 and A157

6. Access function CREATE RETURN CHARGE for the return year and enter the total of boxes A185, A156 and A157 in the field ‘ Notional Tax / Tax Credits’. Do not alter any other amounts

7. Select the [OK] button

Note: If an amendment is made to the 2007/08 or subsequent year’s return and payments on account for the year following the return year are created but are not required under the 80 per cent test, repeat Steps 5 - 7

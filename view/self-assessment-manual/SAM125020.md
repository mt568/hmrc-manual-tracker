This is an historical list where a work item was be created and entered on the ‘Daily Returns Review: Electronic Return Cases’ Work List when a return was filed online, or processed via Rapid Data Capture (RDC), and a coding (integrity) check was required. The reason for the work item will be shown as INT / RDC / PAYE CIC.

No new cases have populated this list since 6 April 2010.

See subject and Action Guide for Daily Returns Review: INT / RDC PAYE CIC (SAM125030).

title: 'Records: agent record: agent maintainer'
---

The role of the SA Agent Maintainer within the Agent Compliance Team is to maintain accurate records of all agents within the SA national database.

The Agent Compliance Team is responsible for

* Setting up new SA agent records
* Amending SA agent records

The SA Agent Maintainer role must be allocated to set up new SA agent codes. This role is only available to the Agent Compliance team.

If an agent’s
name, address or telephone number changes, send details and a copy of any
correspondence to the Agent Compliance Teams mailbox who will arrange for the national agents’ list
to be updated.

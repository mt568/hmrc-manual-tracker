  amend return (Action Guide)'

Where the amended return adds a second or subsequent trading source, to obtain the source reference follow steps 1 - 2 below.

For details of how to access the SA function, select ‘Index Of Functions’ on the left of the screen.

1. Use function MAINTAIN SOURCES and make a note of the source reference for each source. You may want to do so on the relevant pages of the return if so use pencil
2. Enter the source reference during capture of the amended return details in function AMEND RETURN

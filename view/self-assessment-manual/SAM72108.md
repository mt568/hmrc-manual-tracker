The operator with this role is generally responsible for creating a request for a new selection list to select returns for inclusion in the Quality Monitoring Exercise (QME).

This role has the same functions as Clerical Processing Officer ([SAM72082](https://www.gov.uk/hmrc-internal-manuals/self-assessment-manual/sam72082)) plus the extra function SA QME CASE SELECTION to allow the operator to select and view / work the case without having to change role.

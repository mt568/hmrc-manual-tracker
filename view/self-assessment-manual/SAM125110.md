## Husband and wife cases

There is a bulk issue of the non-statutory forms from live PAYE primary source records in April, where the selection criteria is met.

You now have to

* Review the PAYE Service Targeted Review Forms where there is a live PAYE primary source, and update the ‘575 to Husband’ indicator as necessary. More information can be found in the PAYE Manual at PAYE79042
* Prepare a form 575 manually, addressing it to the husband
* If this is done for annual bulk issue, hold the form 575 with other forms to be issued manually

## Civil partnership cases

There is not a bulk issue of the non-statutory forms from live PAYE main source records in April as the ‘575 to Husband’ signal cannot be set in these cases.

You now have to

* Review the PAYE Service Targeted Review Forms where there is a live PAYE primary source. Do not enter the ‘575 to Husband’ indicator. More information can be found in the PAYE Manual at PAYE79042
* Prepare a form 575 manually, addressing it to the civil partner with the PAYE primary source
* If this is done for annual bulk issue, hold the form 575 with other forms to be issued manually

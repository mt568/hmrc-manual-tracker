Note: When the reason for the entry on the ‘Manual Issue of Return’ work list is ‘Welsh Language Signal’, the necessary action to deal with the work item will be taken by the Welsh Contact Centre. Full details for that office are

HMRC Welsh Customer Service  
HM Revenue & Customs  
Thedford House  
High Street  
PORTHMADOG  
LL49 9BF

Telephone number 0300 200 1900

When the Language signal was set to ‘Welsh’, the following action should have been taken

* Where a capacity was present on the taxpayer record

  + Welsh Contact Centre name and address entered as the main capacity address
  + The existing main capacity address entered in the Free Format Notes field in function AMEND TAXPAYER SIGNALS
* Where no capacity was present on the taxpayer record

  + Taxpayer’s name entered in the Communication Name field
  + Welsh Contact Centre name and address entered in the Communication Address field
  + Any existing communication address entered in the Free Format Notes field in function AMEND TAXPAYER SIGNALS

Action to take by the Welsh Contact Centre

* Prepare a Welsh language version of the return manually. Use function MAINTAIN RETURN PROFILE to establish whether a Main Return (SA100) or Short Tax Return (SA200) is required
* If this is done for annual bulk issue, hold the return with other returns to be issued manually
* Prior to issue, use function RECORD DATE OF CLERICAL ISSUE to enter the date of issue on the SA record (the function allows you to enter a date up to 5 days later than today’s date)

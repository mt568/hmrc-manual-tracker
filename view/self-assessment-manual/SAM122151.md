If you have to unlog an unsatisfactory return before capture has taken place follow steps 1 - 6 below. The guide is presented as follows

[Initial action](#IDAKCLAF) - Step 1  
[If the return has been logged](#IDARDLAF) - Step 2  
[If some of the details have been captured](#IDAJFLAF) - Steps 3 - 6

For details of how to access any of the SA functions, select ‘Index Of Functions’ on the left of the screen.

## Initial action

1. Use function MAINTAIN RETURN SUMMARY to

* Check the return has been logged

## If the return has been logged

2. Use function UNLOG RETURN to

* Unlog the return, and
* Select a reason for the unlogging from the drop down menu

## If some of the details have been captured

If some of the details have been captured, using function CAPTURE RETURN

3. Select the Exit option from the File menu heading, you will be prompted to save the return

4. Select the [Delete] button, the return details will be removed from LDC

5. If the whole return is to be sent back cross through any date stamp on the front of the return before returning the form to the nominated partner, see subject ‘Maintain Taxpayer Record: Nominated Partner’ (SAM101290), or agent

6. Keep a copy of the letter you send concerning the unsatisfactory return in the annual run of returns (This content has been withheld because of exemptions in the Freedom of Information Act 2000)
